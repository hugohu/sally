sally中的JavaScript

引入单个插件还是一次性引入所有插件

每个插件都可以单独的引入到页面中（注意插件间的依赖关系），或者一次性引入。 sally.js 和sally.min.js 文件都将所有插件包含在一个文件中了（前者是未压缩版，后者是压缩版）。

Data 属性

你可以仅仅通过data属性API就能使用所有sally中的插件，而且不用写一行JavaScript代码。这是sally中的一等API，并且是你的首选方式。

data api的使用方法:对需要执行事件的对象添加自定义属性 data-event 定义事件类型

sally的插件部分主要放置了一些常用的有特定场景的需要重新制作.可以通过更多来进行添加插件,全局定义的选中样式名称为active 所以注意不要覆盖这个样式名,并且通过层级来实现特定的效果

滑动门

查看案列: 

<dl class="m-tab" data-event="tab">
  <dt class="m-hd"> 
    <ul>
      <li class="active"><a href="#">默认：居左，不定宽</a></li>
      <li><a href="#">“标题”和“更多”可删</a></li>
      <li><a href="#">扩展类可以自由组合</a></li>
      <li><a href="#">标签标签</a></li>
    </ul> 
</dt>
  <dd class="active">1 </dd>
  <dd>   2</dd>
  <dd>   3</dd>
  <dd>   4</dd>
</dl>

通过一个dl列表,dt里定义ul切换对象的下面切换,这种结构是最简介的方式,dt部分可以参考tab模块进行修改添加

如果要转换成个点击事件,可以添加data-type="click" 来进行转行,例如

<dl class="m-tab" data-event="tab" data-type="click" >

图片懒加载

这种处理方式主要是为了减少链接并发数,图片的src地址初始状态放置一张临时图片,然后data-url的自定义属性里放着真实的地址.跟随滚动条动态加载图片

<img alt="博洛尼" src="http://meta.bmlink.com/images/d.png" data-url="http://img1.bmlink.com/customizepage/2013/5/15/9/280303170103721.gif" height="70" width="185" />

返回顶部

添加自定义属性 data-event="gotop" 返回顶部隐藏的效果,具体的html可以自己灵活的进行拼接,比如增加评测或者其他的模块,定位信息在css里编写

<a href="javascript:" id="gotop" data-event="gotop" title="&#36820;&#22238;&#39030;&#37096;"></a> 


/*css部分*/

  opacity: .7;filter: alpha(opacity=70);

}
#gotop:hover {
opacity:1;filter: alpha(opacity=1);
