焦点图小标样式

html格式统一为div标签里直接嵌套a标签,通过修改样式达到不同的效果,可以自定义属性添加也可以采用默认的

<div class="u-dot">
<a title="1" class="active" href="#">1</a>
<a title="2" href="#" class="">2</a>
<a class="" title="3" href="#">3</a>
<a title="4" href="#">4</a>
</div>

默认样式

1234
.u-dot{width: 100%; text-align: right; background:#000; position:absolute; bottom:0; right:0; height:20px;z-index:2;background:rgba(0,0,0,0.5);filter:progid:DXImageTransform.Microsoft.Gradient(GradientType=0, StartColorStr='#4c000000', EndColorStr='#4c000000')}

.u-dot a {text-indent: -999px; display: inline-block; cursor:pointer;line-height:15px; text-align:center; font-size:12px; width:25px; height:10px; cursor:pointer; overflow:hidden; background:#919191; margin:5px 4px 0 0; filter:alpha(opacity=80); -moz-opacity:.8; opacity:.8; -webkit-transition:All .5s ease;-moz-transition:All .5s ease;-o-transition:All .5s ease}

.u-dot a.active { background:#fff; filter:alpha(opacity=100); -moz-opacity:1; opacity:1 }
